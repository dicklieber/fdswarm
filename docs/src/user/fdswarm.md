# FDSwarm 

FdSwarnm is an amateur radio logging application specifically designed for ARRL and Winter Field Days.

- Cross-platform compatibility: FDSwarm supports macOS, Microsoft Windoes and Linux, ensuring accessibility for a wide range of users.
- Automatic network discovery: Detects and syncronizes QSO with other FDSwarm instances on the local network
- Nodes can come and go dynamically, ensuring that the swarm remains resilient and adaptable to changing network conditions.
- Generate Cabrillo files for contest submission.
